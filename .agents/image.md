# 각 image assets들이 어떻게 사용되는지 기록

## assets : 아이콘 폴더
- logo.png : 어플 로고 이미지
- leftlogo : 좌측 위 로고 
- settings : 메인 화면 우측 위 환경설정 아이콘 4개 일괄
- speakbtn : 작업일지 번역시 눌러서 말하기 버튼
- translate_speak.png: 실시간 번역기에서 눌러서 말하기 버튼(글자,아이콘 포함)
- emergency: 긴급사항에서 상단 현장공, 구급구조, 시설보안 버튼 3개 일괄
- profile : 앱 내 쓰이는 모든 기본 프사
- today: 안전점검 상단 오늘의점검 패널 (이미지, 아이콘 포함)
- status : 이수현황 패널 (파란 부분 글자,아이콘 모두 포함)



## assets/nav : 하단바 고정아이콘
### _Act는 각각의 아이콘의 활성화 버전이다.
- icon0 : 3번째 아이콘 (관리자기능)
- icon1 : 1번째 아이콘 (번역)
- icon2 : 2번째 아이콘 (채팅)
- icon3 : 3번째 아이콘 (작업일지)
- icon4 : 4번째 아이콘 (안전점검)
